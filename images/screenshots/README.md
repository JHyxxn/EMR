# 스크린샷 이미지 폴더

이 폴더에는 보고서에 사용할 스크린샷 이미지들이 저장됩니다.

## 이미지 파일 명명 규칙

- `01-login.png` - 로그인 화면 (Dr.App • EMR, 로그인 버튼)
- `02-dashboard.png` - 홈 대시보드 (금일 대기 환자 목록, 병원 일정, 처방/오더보드)
- `03-new-patient.png` - 신규 환자 등록 (환자 기본 정보, 바이탈 입력 폼)
- `04-patient-chart-exam.png` - 진료 차트 (이지현 환자, 이전 방문 내역, SOAP 기록 작성, AI 정리 버튼, 검사 탭)
- `05-exam-results.png` - 검사 결과 (검사 탭, CT 검사 및 혈액검사 결과, 이상 수치 빨간색 강조)
- `06-medical-opinion.png` - 소견서 발급 모달 (소견서 발급 창, 환자 정보, 진료 내용 자동 요약)
- `07-revisit-chart.png` - 재진환자 차트 (재진환자 목록, 이전 기록, 대기 목록 추가)
- `08-document-management.png` - 서식/문서 관리 (환자 목록, 발급된 소견서 목록)

## 이미지 추가 방법

1. 스크린샷을 캡처합니다
2. 파일명을 위의 명명 규칙에 맞게 저장합니다
3. 이 폴더에 이미지 파일을 저장합니다
4. Git에 커밋 및 푸시합니다

## 권장 이미지 형식 및 크기

- **형식**: PNG 또는 JPG
- **픽셀 크기**: 
  - 권장: 1920 x 1080 (Full HD, 16:9 비율)
  - 최소: 1600 x 900
  - 최대: 2560 x 1440 (2K)
- **파일 크기**: 각 이미지당 2MB 이하 권장
- **비율**: 16:9 또는 4:3 (일관성 유지)

## 보고서 삽입 시 크기

보고서에 삽입할 때는 다음과 같이 설정하세요:
- **너비**: 16cm (A4 용지 기준)
- **높이**: 자동 조정 (원본 비율 유지)
- 또는 **너비**: 100% (페이지 너비에 맞춤)
